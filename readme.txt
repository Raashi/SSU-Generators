Версия интерпретатора 3.6. Список необходимых библиотек в source\requirements.txt
Файл source\gen.py - генераторы
Файл source\dist.py - распределения
Файл source\analysis.py - лабораторная работа

Все эти скрипты также скомпилированы в исполняемые файлы:
\release\<gen/dist/analysis>.exe
Скомпилированные exe работают только внутри каталога release.

Каталог lab содержит результаты экспериментов, представленные в лабораторной работе.
